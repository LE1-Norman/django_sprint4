# django_sprint4

Blogicum - это полнофункциональная блог-платформа, разработанная на Django, которая позволяет пользователям создавать, публиковать и комментировать записи. Проект включает все основные функции современного блога.

## Основные возможности
- Аутентификация и авторизация пользователей

Приложение включает такие компоненты как: регистрация и аутентификация пользователей, личный кабинет пользователя, редактирование профиля и смена пароля, а также разграничение прав доступа к различным сервисам.

- Работа с публикациями

В приложении реализованы следующие методы работы с публикациями: Создание постов, а также их редактирование и удаление, есть возможность прикрепления изображений к постам. Автор может воспользоваться отложенной публикацией постов (планирование постов). Все посты разделены по категориям и местоположению, которые в свою очередь управляются администратором.
Для удобства пользователей была применена пагинация (10 постов на странице).

- Работа с комментариями

В приложение включена система комментирования постов, которая позволяет авторизированным пользователям писать комментарии, редактировать и удалять их.
Комментарии сортируются по времени публикации, а для каждого поста происходит подсчет количества комментариев.

- Дополнительно

В проекте реализованы статичные страницы, кастомные страницы ошибок и отправка email-сообщений (файловый бэкенд).

## Настройка и запуск
1) Клонируйте репозиторий и перейдите в него в командной строке:
   ```bash
   git clone https://github.com/LE1-Norman/django_sprint4.git
   cd django_sprint4
   ```
2) Создайте и активируйте виртуальное окружение
   ```bash
   python -m venv venv
   ```
   source venv/Scripts/activate  # Windows (Git Bash)
3) Установите зависимости
   ```bash
   pip install -r requirements.txt
   ```
4) Выполните миграции
   ```bash
   python manage.py makemigrations
   python manage.py migrate
   ```
5) Запустите проект
   ```bash
   python manage.py runserver
   ```
